# 快速開始指南

## 🚀 最簡單的方式：一鍵啟動

### 只需要一個指令！

```bash
./code/streamlit_app/run_app.sh
```

**就這麼簡單！** 腳本會自動：
- 檢查並建立虛擬環境（如果需要）
- 安裝所有必要套件
- 啟動應用程式

---

## 📋 詳細步驟（如果你想了解細節）

### 步驟 1：確認環境

確保已安裝 Python 3.8+：

```bash
# 檢查 Python 版本
python3 --version
```

**注意**：不需要手動安裝套件！啟動腳本會自動在虛擬環境中安裝。

### 步驟 2：啟動應用程式

使用啟動腳本（推薦）：

```bash
cd code/streamlit_app
./run_app.sh
```

或直接使用 Python：

```bash
cd code/streamlit_app
python3 -m streamlit run taiwan_lda_explorer.py
```

### 步驟 3：開始探索

瀏覽器會自動開啟 `http://localhost:8501`，你就可以開始使用了！

---

## 📱 應用程式功能快覽

### 側邊欄設定
- **主題數 (K)**：調整 LDA 模型的主題數量（3-10）
- **分析模組**：選擇不同的分析頁面

### 四大功能模組

#### 1️⃣ 主題總覽
- 查看所有主題的關鍵詞
- 切換表格/視覺化顯示模式
- 調整顯示的關鍵詞數量

#### 2️⃣ 醫院比較
- 選擇 2-5 家醫院進行比較
- 長條圖/熱力圖視覺化
- 匯出詳細數值表格

#### 3️⃣ 主題深入探索
- 選擇特定主題進行分析
- 查看關鍵詞權重
- 顯示詞雲圖

#### 4️⃣ 統計儀表板
- 資料集總體統計
- 各醫院評論數量分佈
- LDA 分析結果視覺化

---

## 💡 使用技巧

### 切換主題數
在側邊欄拖動滑桿即可切換不同的 K 值（需要事先訓練對應的模型）

### 醫院比較
建議選擇 3-5 家醫院，太多會讓圖表難以閱讀

### 匯出圖表
在圖表上按右鍵可以儲存為圖片

### 效能優化
首次載入會快取資料，之後會更快。如需重新載入，點擊右上角的重新整理按鈕

## ❓ 常見問題

**Q: 找不到 LDA 模型檔案**
- 確認 `results/taiwan_lda_k7/` 目錄下存在模型檔案
- 預設使用 K=7，如要使用其他 K 值需先訓練模型

**Q: 醫院比較顯示示範資料**
- 目前版本使用示範資料
- 完整版需要個別醫院的評論標記

**Q: 如何停止應用程式**
- 在終端機按 `Ctrl+C`

**Q: 應用程式沒有自動開啟瀏覽器**
- 手動開啟瀏覽器並前往 `http://localhost:8501`

**Q: 看到 numpy 或套件導入錯誤**
- 使用啟動腳本，它會自動在虛擬環境中執行
- 虛擬環境避免了套件衝突問題

**Q: 想了解虛擬環境？**
- 查看專案根目錄的 `VENV_GUIDE.md`
