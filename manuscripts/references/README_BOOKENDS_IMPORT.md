## 檔案說明

---

## 匯入到 Bookends 的步驟

### 方法 1：直接匯入 RIS 檔案（推薦）

1. **開啟 Bookends**
   - 啟動 Bookends 應用程式

2. **匯入檔案**
   - 選擇選單：`File` → `Import` → `Import References`
   - 或使用快捷鍵：`⌘I`

3. **選擇檔案**
   - 瀏覽到此目錄
   - 選擇 `all_references.ris`
   - 點擊 `Open`

4. **確認匯入格式**
   - Bookends 會自動識別 RIS 格式
   - 如果需要手動選擇，請選擇 `RIS` 作為匯入格式

5. **完成匯入**
   - 點擊 `Import` 按鈕
   - 等待匯入完成（73 筆文獻）
   - 檢查匯入的文獻是否正確

### 方法 2：拖放匯入

1. 在 Finder 中找到 `all_references.ris` 檔案
2. 直接拖放到 Bookends 的主視窗
3. Bookends 會自動匯入文獻

## 常見問題

### Q1: 匯入後發現重複文獻怎麼辦？
**A**: 使用 Bookends 的 `References` → `Find Duplicates` 功能來識別和合併重複項。

### Q2: 某些欄位是空白的？
**A**: 這是正常的。有些文獻（如報告、書籍）可能沒有卷期號等資訊。

### Q3: 可以匯入到其他文獻管理軟體嗎？
**A**: 可以！RIS 格式是標準格式，以下軟體都支援：
- EndNote
- Zotero
- Mendeley
- RefWorks
- Papers

### Q4: 如何匯出成其他格式？
**A**: 在 Bookends 中：
1. 選擇所有匯入的文獻
2. `File` → `Export` → `Export References`
3. 選擇目標格式（BibTeX, EndNote XML, etc.）
