# 台灣與美國醫院評論跨文化比較分析報告
## Taiwan-USA Hospital Reviews Cross-Cultural Comparison Analysis

**分析日期**: 2025年10月30日
**分析方法**: Gensim LDA (K=7)
**資料集**: 台灣5,007筆評論 | 美國3,240筆評論

---

## 一、執行摘要 | Executive Summary

本研究使用相同的Gensim LDA方法論，以7個主題分析台灣與美國頂尖醫院的病患評論，揭示兩地醫療服務品質的關注焦點差異。

### 關鍵發現 | Key Findings

1. **模型品質**：台灣模型一致性較高（0.4175 vs 0.3887），顯示台灣病患評論在主題聚焦度上更為集中
2. **評論分布**：兩國皆有明顯的正面主題（台灣主題1佔24.9%，美國主題5佔37.1%）
3. **文化差異**：
   - 台灣更關注**服務態度**（主題3，17.3%）
   - 美國更突出**疼痛管理**（主題7，19.9%）和**等待時間**（主題2，13.9%）
   - 美國出現**帳務問題**主題（主題6，4.1%），台灣無此類主題

### 解讀 | Interpretation

- **台灣模型**一致性分數較高，表示主題內部詞彙語義關聯性更強，可能反映文化同質性
- **美國模型**困惑度較低（絕對值較小），表示模型預測能力稍佳
- 兩個模型品質皆屬可接受範圍（Coherence > 0.35）

### 獨特主題 | Unique Topics

#### 台灣獨有：
- **主題3**（服務態度問題，17.3%，1.98★）
  - 關鍵詞：態度、病人、護理師、人員、不是
  - **文化意涵**：台灣病患對服務態度高度敏感，"態度"問題佔相當大比例
  - 這反映東亞文化中對人際互動品質的重視

- **主題4**（設施與便利性，10.2%，2.76★）
  - 關鍵詞：停車場、方便、動線、電梯
  - **文化意涵**：台灣病患關注醫院物理環境的便利性

#### 美國獨有：
- **Topic 7**（疼痛管理問題，19.9%，1.92★）
  - 關鍵詞：pain, day, nurse, time, told
  - **文化意涵**：美國病患對疼痛管理有明確期待，疼痛控制不佳是重要負面因素
  - 可能與美國鴉片類止痛藥管制政策變化有關

- **Topic 6**（預約與帳務，4.1%，2.72★）
  - 關鍵詞：appointment, daughter, child, time, service
  - **文化意涵**：美國醫療系統中帳務、預約、保險等行政流程是獨特痛點
  - 台灣健保制度下此類問題較少出現

## 九、技術細節 | Technical Details

### 資料前處理 | Data Preprocessing

**台灣（中文）**：
- 使用jieba分詞
- 移除停用詞（的、是、在等）
- 自訂醫療停用詞（醫院、看病等）
- 詞頻過濾：no_below=3, no_above=0.5

**美國（英文）**：
- 使用NLTK tokenization + lemmatization
- 移除英文停用詞
- 自訂醫療停用詞（hospital, doctor, went等）
- 詞頻過濾：no_below=3, no_above=0.5

### 模型參數 | Model Parameters

```python
LdaModel(
    corpus=corpus,
    id2word=dictionary,
    num_topics=7,
    alpha='symmetric',
    eta='auto',
    iterations=100,
    passes=10,
    random_state=42
)
```

### 評估指標 | Evaluation Metrics

- **Coherence Score (c_v)**：測量主題內部詞彙語義一致性（越高越好，>0.35可接受）
- **Perplexity**：測量模型預測能力（越低越好，負值的絕對值越小越好）
